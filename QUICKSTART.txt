## вҡҷпёҸ API Endpoints

### Health Check
```
GET http://127.0.0.1:8001/health
```

### Upload Image
```
POST http://127.0.0.1:8001/upload-form
Body: multipart/form-data with 'photo' field
Response: JSON with matches and confidence scores
```

### Embed Image
```
POST http://127.0.0.1:8001/embed
Body: multipart/form-data with 'photo' field
Response: JSON with 512-dim embedding vector
```

### Get Neighbors
```
POST http://127.0.0.1:8001/nn?k=5
Body: multipart/form-data with 'photo' field
Response: Top-k similar images with distances
```

## рҹ”§ Troubleshooting

### Backend Won't Start
```powershell
# Kill any existing Flask process
Get-Process python | Stop-Process -Force

# Try again
.\start_all.bat
```

### Can't Access Frontend
- Check URL: `http://127.0.0.1:8080/main.html` (not `localhost`)
- Check both services are running
- Clear browser cache (Ctrl+Shift+Del)

### Upload Fails with 404
- Backend crashed - check logs
- Try restarting with `.\start_all.bat`
- Check gallery folder exists

### Slow Image Processing
- First run is slower (model initialization)
- Check CPU usage (should spike during processing)
- On first upload, wait up to 10 seconds

## рҹҶҳ Emergency Stop

### Stop Backend
- Close Flask terminal window
- Or run: `Get-Process python | Stop-Process -Force`

### Stop Frontend  
- Close http-server terminal window
- Or: `Get-Process node | Stop-Process -Force`

### Hard Reset
```powershell
# Kill all Python and Node processes
Get-Process python, node -ErrorAction SilentlyContinue | Stop-Process -Force

# Start fresh
.\start_all.bat
```
